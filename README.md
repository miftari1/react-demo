This is a demo project building a Tic-Tac-Toe game using React.
It includes gameboard with 9 fields where each player take turns writing either X or O until one player fills three consecutive boxes vertically, horizontally or diagonally. The game includes also a history field, where each player can click on past move so the app renders this specific state of the game and a message field for current player and result of the game.
You can start the project using **npm start** command.
